# Go Bilibili Helper

Inspired by [metowolf/BilibiliHelper](https://github.com/metowolf/BilibiliHelper).

## Usage

Edit `config.json`

```json
{
  "Username": "username",
  "Password": "password"
}
```

Build and run

```bash
go build
go-bilibili-helper.exe 23058 config.json
```
